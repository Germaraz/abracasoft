Gestión de clientes

En este módulo usted podra, crear editar o eliminar los clientes de su negocio. Puede comenzar dando el alta de clientes con solo presionar el boton "Nuevo cliente", tras esto se abrirá la venta con los campos a rellenar.

Nota: el campo codigo postal se rellenará automagicamente tras seleccionar la localidad

Una vez completado los datos del cliente y presionando el boton "Guardar", se guardarán los datos del cliente y el mismo estará disponible para su búsqueda en la ventana del modulo. Seleccionando el cliente puede editar sus propiedades y eliminarlo, presionando los botones "Editar cliente" o "Eliminar cliente" respectivamente.

Una vez creado el cliente este se mostrará en la ventana principal, en el sector izquierdo se visualizarán los clientes que hayan sido cargados y, en el sector derecho las ventas que se hicieron a estos y el monto de un pago.

Asimismo, el sistema contempla a los clientes en estado moroso, en el area derecha de la ventana se mostrará un apartado con el monto de las ventas hechas, la fecha de las mismas y el pago que realizó oportunamente, esta acción puede realizarse con el botón "Agregar pago", el cúal tras ser presionado accionará una ventana que permitirá agregar el monto del pago.
