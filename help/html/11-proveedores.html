Gestión de proveedores

En este modulo usted podra gestionar los proveedores de productos.

Podra, crear editar o eliminar los proveedores de su negocio. Puede comenzar dando el alta de proveedores con solo presionar el boton "Nuevo proveedor", tras esto se abrirá la venta con los campos a rellenar.

Nota: el campo codigo postal se rellenará automagicamente tras seleccionar la localidad

Una vez completado los datos del proveedor y presionando el boton "Guardar", se guardarán los datos del proveedor y el mismo estará disponible para su búsqueda en la ventana del modulo. Seleccionando el proveedor puede editar sus propiedades y eliminarlo, presionando los botones "Editar cliente" o "Eliminar cliente" respectivamente.

Una vez creado el proveedor este se mostrará en la ventana principal, en el sector izquierdo se visualizarán los proveedores que hayan sido cargados y, en el sector derecho las compras que se hicieron a estos y el monto de un pago.

Asimismo, el sistema contempla el hecho que haya quedado un saldo pendiente de pago al proveedor, en el area derecha de la ventana se mostrará un apartado con el monto de las ventas hechas, la fecha de las mismas y el pago que realizó oportunamente, esta acción puede realizarse con el botón "Agregar pago", el cúal tras ser presionado accionará una ventana que permitirá agregar el monto del pago.
